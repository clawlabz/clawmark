## 3. Design System (extracted from Stitch)

### Colors
- **Primary accent**: `#137fec` (blue)
- **Dark mode background**: dark neutral (extracted from Stitch dark theme)
- **Light mode background**: light neutral (extracted from Stitch light theme)
- **Text primary**: white (dark mode) / dark gray (light mode)
- **Text muted**: gray for URLs and descriptions
- **Tag chips**: colored badges with accent variations

### Typography
- **Font family**: Inter
- **Heading**: Inter Bold
- **Body**: Inter Regular
- **Muted text**: Inter Regular, smaller size, gray

### Spacing & Layout
- **Roundness**: 8px border radius on cards, buttons, inputs
- **Desktop layout**: Fixed sidebar (tags) + fluid main content
- **Mobile layout**: Full-width, horizontal scrollable tag strip, FAB for add
- **Card spacing**: Consistent padding between bookmark cards

### Theme Strategy
- Auto-detect via `prefers-color-scheme`
- Dark mode: primary experience (matches developer preference)
- Light mode: supported for system-light users
- Tailwind `dark:` classes with `class` strategy
